Менеджер Подписок

Удобное браузерное расширение для контроля всех ваших подписок в одном месте**

Возможности

- Автоматический расчёт расходов** с поддержкой нескольких валют (₽, $, €, £, ₸)
- Умные уведомления** за 3 дня, 1 день и в день платежа
- Светлая и тёмная тема** оформления
- Бейдж** с суммой расходов на иконке расширения
- Экспорт / Импорт** данных в JSON
- Быстрая сортировка** по дате ближайшего платежа
- Полная конфиденциальность** — все данные хранятся только на вашем устройстве

Установка

1. Скачайте архив `subscription-extension.zip`
2. Распакуйте его в удобную папку
3. Откройте в браузеере страницу: `chrome://extensions/`
4. Включите «Режим разработчика»
5. Нажмите «Загрузить распакованное расширение»
6. Выберите папку `subscription-extension`
7. Готово!

 Как пользоваться

1. Нажмите на иконку расширения в панели браузера
2. Нажмите **«+ Добавить подписку»**
3. Начните вводить название сервиса — появится автодополнение
4. Укажите стоимость и дату следующего платежа
5. Готово! Расширение будет напоминать о платежах автоматически
